# web_link

Web linking header parser library for [Pony](https://www.ponylang.io/).

## Status

Under development. The API is not yet stable.

## Installation

* Install [corral](https://github.com/ponylang/corral)
* `corral add github.com/ponylang/web_link.git --version 0.0.0`
* `corral fetch` to fetch your dependencies
* `use "web_link"` to include this package
* `corral run -- ponyc` to compile your application
